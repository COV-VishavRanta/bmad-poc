### ESLint Configuration
Create `.eslintrc.json` with Next.js and TypeScript rules:
```json
{
  "extends": [
    "next/core-web-vitals",
    "@typescript-eslint/recommended",
    "prettier"
  ],
  "rules": {
    "@typescript-eslint/no-unused-vars": "error",
    "@typescript-eslint/no-explicit-any": "warn",
    "react-hooks/exhaustive-deps": "warn"
  }
}
```

### Prettier Configuration
Create `.prettierrc` with consistent formatting:
```json
{

  "singleQuote": true,
  "printWidth": 80,
  "tabWidth": 2,
  "useTabs": false
}
```

### Python Tool Configuration
Create `pyproject.toml` with tool settings:
```toml
[tool.black]
line-length = 88
target-version = ['py311']

[tool.isort]
profile = "black"
multi_line_output = 3

[tool.mypy]
python_version = "3.11"
strict = true
```


Backend development commands (Makefile):
```makefile
.PHONY: dev lint format test migrate

dev:
	uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

lint:
	flake8 app/
	mypy app/

format:
	black app/
	isort app/

test:
	pytest app/tests/

migrate:
	alembic upgrade head
```

### Environment Setup Instructions
Steps for new developer onboarding:
1. Clone repository and install Docker/Docker Compose
2. Copy `.env.example` to `.env` with local values
3. Run `docker-compose up` to start all services
4. Install frontend dependencies: `cd frontend && npm install`
5. Install backend dependencies: `cd backend && pip install -r requirements.txt`
6. Run database migrations: `make migrate`
7. Install pre-commit hooks: `pre-commit install`
