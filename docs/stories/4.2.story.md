### Form Validation Schema
Project form validation using Zod:
```typescript
const projectFormSchema = z.object({
  name: z.string().min(1, 'Project name is required').max(255),
  clientId: z.number().min(1, 'Client selection is required'),
  sowId: z.number().min(1, 'SOW selection is required'),
  projectType: z.enum(['Development', 'Maintenance', 'Consulting', 'Support']),
  status: z.enum(['planned', 'active', 'on_hold', 'completed', 'cancelled']),
  startDate: z.date(),
  endDate: z.date(),
  groupId: z.number().optional(),
  description: z.string().max(1000).optional(),
}).refine((data) => data.startDate <= data.endDate, {
  message: "End date must be after start date",
  path: ["endDate"],
})
```

### API Integration
[Source: docs/stories/4.1.story.md#api-endpoint-specifications]
Frontend API calls for project management:
- **GET /api/projects**: Fetch projects with search and pagination
- **POST /api/projects**: Create new project
- **GET /api/projects/{id}**: Fetch single project with details
- **PUT /api/projects/{id}**: Update project information
- **PUT /api/projects/{id}/status**: Change project status
- **DELETE /api/projects/{id}**: Delete/archive project
