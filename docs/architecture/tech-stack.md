# Technology Stack

## Frontend Technologies

### Core Framework
- **Next.js 15+**: React framework with App Router for server-side rendering and static generation
- **TypeScript**: Strongly typed JavaScript for better development experience
- **React 18+**: Component-based UI library

### Styling and UI
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development
- **Material-UI (MUI) v5**: React component library for consistent design system
- **CSS Modules**: For component-scoped styling when needed

### State Management and Data Fetching
- **React Query (TanStack Query)**: Server state management and caching
- **Zustand**: Client-side state management for global application state
- **React Hook Form**: Form state management and validation

## Backend Technologies

### Core Framework
- **FastAPI**: Modern Python web framework for building APIs
- **Python 3.11+**: Programming language
- **Pydantic**: Data validation and settings management using Python type annotations

### Database and ORM
- **MySQL 8.0+**: Relational database
- **SQLAlchemy 2.0+**: Python SQL toolkit and Object-Relational Mapping
- **Alembic**: Database migration tool for SQLAlchemy

### Authentication and Security
- **Passlib**: Password hashing library
- **python-jose**: JWT token handling
- **Session-based authentication**: Using HttpOnly cookies for security

## Development and Deployment

### Containerization
- **Docker**: Containerization platform
- **Docker Compose**: Multi-container application orchestration

### Development Tools
- **ESLint**: JavaScript/TypeScript linting
- **Prettier**: Code formatting
- **Black**: Python code formatting
- **Flake8**: Python linting

### Testing
- **Jest**: JavaScript testing framework
- **React Testing Library**: React component testing
- **Pytest**: Python testing framework
- **Supertest**: API testing for Python

## Environment Requirements

### Node.js (Frontend)
- Node.js 18+ LTS
- npm or yarn package manager

### Python (Backend)
- Python 3.11+
- pip package manager
- Virtual environment (venv or conda)

### Database
- MySQL 8.0+
- Database client tools (MySQL Workbench, phpMyAdmin, or CLI)

### Docker
- Docker Engine 20.10+
- Docker Compose v2
